## 一、摘要（30秒版）

**问题1问的是：** 药物C（对照组）、药物A、药物B这三组受试者，入组时的基本情况（年龄、婚姻状况、既往用药史、初始抑郁程度）是否本来就存在差异？

**为什么这一步必须先做：** 问题2、3要比较三种药物的疗效好坏。但如果三组人一开始底子就不一样（比如某一组本来病情更重、年纪更大），那后面看到的"疗效差异"就可能是人群本身的差异造成的，而不是药物的问题。这一步就是排除这个干扰。

**结论：**

| 基线变量 | 三组之间有无显著差异 | 说明 |
|---|---|---|
| 年龄 | **无差异** | 三组年龄分布几乎一致 |
| 既往用药史 | **无差异** | 三组用药史构成一致 |
| 初始抑郁程度 | **无差异** | 三组病情轻重构成一致 |
| 婚姻状况 | **表面有差异，经排查非真实差异** | 总卡方的 **91.5%** 由"未知"一列贡献（该列 70 人全来自药物B组）；剔除缺失后 χ²=9.810, **p=0.133 不显著**，Cramér's V 从 0.156 降到 0.040 → 差异是缺失制造的 |

**一句话结论：** 三组在年龄、既往用药史和初始抑郁程度上的差异很小；婚姻状况的总体卡方有 91.5% 来自 B 组70人的集中缺失，剔除未知后 p=0.133、Cramér's V=0.040。现有数据只能说明**缺失率与组别强相关，可能存在记录流程异常**，不能仅凭这一点判定为 MNAR，也不能直接写成三组婚姻构成存在实质差异。

## 二、数据说明

数据来自附件1（两个医院合并后共 3149 人：一院1575人 + 二院1574人，理论上应各525人×3组×2医院=3150人，二院组1实际只有524人，少1人，属于数据本身的情况，不是我们处理错了）。

附件1里每个受试者一行，关键列是"one-hot"编码（下面举例说明，方便大家看代码时理解）：

```
婚姻状况：未婚 已婚 离异 丧偶   ← 4列，命中的那一列填1，其余空着
既往抗抑郁药使用情况：无 使用过抗抑郁药 其它   ← 3列
抑郁程度：轻度 中度 重度   ← 3列
```

代码里第一步做的事情，就是把这种"4列拆成1列"的编码，还原成 1 个正常的分类变量（比如把"未婚=1,已婚=空,离异=空,丧偶=空"这一行还原成"婚姻状况=未婚"）。**如果4列全是空的（说明这项没记录/失访），我们没有直接删掉这个人，而是记成"未知"这个类别**，这样既不会丢样本量，又能如实反映缺失的情况。

## 三、方法怎么选的（给想看细节 / 要写论文方法部分的同学）

### 3.1 年龄（连续变量）——三组比不比得动？

年龄是数字，要比较3组的均值/分布是否一样，常规套路是"方差分析 ANOVA"，但ANOVA有个前提：每组数据要接近正态分布，且各组方差要差不多。所以流程是：

1. 先用 **Shapiro-Wilk 正态性检验** 看每组年龄是否正态
2. 再用 **Levene 方差齐性检验** 看三组方差是否相近
3. 如果两个前提都满足 → 用 **单因素方差分析 ANOVA**
4. 如果任一前提不满足 → 改用不要求正态分布的 **Kruskal-Wallis 检验**（ANOVA的"非参数版"，比较的是排序后的分布而不是均值本身）

> 我们这批数据每组n=1000+，Shapiro检验在这种大样本下几乎总会判定"不服从正态"（这是这个检验本身的已知特性，样本量一大就很敏感），所以代码自动切换到了 Kruskal-Wallis，结果 p=0.637（远大于0.05），**说明三组年龄分布没有显著差异**。

如果结果显著，代码里还准备了事后两两比较（ANOVA显著就用Tukey HSD，Kruskal-Wallis显著就用Mann-Whitney U两两比较+Bonferroni校正），但这次年龄没有触发，用不上。

### 3.2 婚姻状况 / 既往用药 / 抑郁程度（分类变量）——用卡方检验

这三个都是"组别 × 类别"的列联表问题（比如"3组 × 婚姻状况4类"），标准方法是 **卡方独立性检验（chi-square test of independence）**：检验"落入哪个类别"跟"属于哪一组"是不是没关系（原假设：无关，即三组分布一致）。

卡方检验有个使用条件：列联表里每个格子的**期望频数不能太小（一般要求≥5）**，否则近似不准。代码里对每个变量都检查了这一点，如果有格子期望频数<5，会额外跑一个 **蒙特卡洛置换检验**（把组别标签随机打乱2000次，看真实卡方值排第几）作为交叉验证，这样即使卡方检验的前提不完全满足，结论也站得住。

### 3.3 多重比较校正

我们一次性测了4个基线变量（年龄、婚姻、用药史、抑郁程度），每个都用 α=0.05 的标准判断"显著"，但测的次数一多，本来不显著的东西"蒙对"显著的概率也会累积升高（多重比较问题）。所以代码额外做了一次 **FDR (Benjamini-Hochberg) 校正**，把4个p值一起调整，校正后的显著性判断更可靠，这也是我们最终结论表里用的判断标准。

### 3.4 婚姻状况显著的原因排查

卡方检验发现婚姻状况在三组间显著（χ²=152.871, p<0.001），但看列联表（`output/test_results.csv`）会发现：药物C组和药物A组的"未知"人数都是0，唯独药物B组有70个人婚姻状况缺失。为确认这个显著性到底是不是缺失造成的，我们做了三步定量排查（不是"看上去差不多"，是算出来的）：

**第一步：效应量对不上。** Cramér's V = 0.156，按 Cohen 标准（df\*=2 时 small=0.07/medium=0.21/large=0.35）只是"小到中等"，跟 p≈5×10⁻²⁹ 这种"极度显著"完全不匹配。这是大样本（N=3149）下卡方检验过度敏感的典型信号 —— **统计显著 ≠ 实质差异**，必须往下查。

**第二步：卡方贡献度分解 —— 显著性是哪一列贡献的。** 把总卡方按列拆开（每格贡献 = (O−E)²/E）：

| 类别 | 对卡方的贡献 | 占比 |
|---|---|---|
| **未知** | **139.93** | **91.5%** |
| 已婚 | 4.63 | 3.0% |
| 丧偶 | 3.71 | 2.4% |
| 离异 | 3.07 | 2.0% |
| 未婚 | 1.54 | 1.0% |

**总卡方的 91.5% 由"未知"一列单独贡献**，四个真实婚姻类别加起来只占 8.5%。

**第三步：剔除"未知"后重新检验 —— 差异是否消失。**

| 方案 | χ² | df | p | Cramér's V | 结论 |
|---|---|---|---|---|---|
| 含"未知" | 152.871 | 8 | <0.001 | 0.156 | 显著 |
| **剔除"未知"** | **9.810** | **6** | **0.133** | **0.040** | **不显著** |

剔除后三组构成（已婚/未婚/离异/丧偶）分别为 53.5%/35.4%/9.0%/2.2%（组1）、54.0%/35.8%/7.0%/3.1%（组2）、51.3%/35.2%/9.6%/3.9%（组3，n=980），效应量近乎归零。

**第四步：描述缺失模式。** 单独拿"是否缺失 × 组别"做 3×2 检验：χ²=143.115, df=2, p=8.4×10⁻³²，说明缺失率与组别强相关，因此不符合简单的 MCAR 图景。但 MNAR 要求缺失概率依赖未观测到的真实婚姻状态，现有数据无法检验；这里统一称为**组间差异性缺失/可能的记录流程异常**。

**结论：婚姻状况的组间"差异"完全由缺失分布不均制造，不是三组婚姻构成本身不同。** 所以报告里要写"该差异由B组数据缺失所致"，**不能**写"三组患者婚姻状况本身不一样"。这是数据记录层面的问题，需作为局限性说明。

> 复现方式：这四步排查的代码见 `docs/论文初稿.md` 附录 C.5（可直接粘贴运行，不依赖 `problem1_baseline.py`）。

我们还顺手检查了"医院分配是否均衡"（会不会某个医院偏向收某一组病人），卡方 p=0.9997，**完全均衡**，排除了这个额外的干扰因素。

## 四、代码结构（方便接手 / 复用）

```

├── setup_cjk_font()        中文字体注册（画图用，避免出现方框乱码）
```

## 六、写论文/汇报时可以直接用的话术

> 采用卡方独立性检验（分类变量）和 Kruskal-Wallis 秩和检验（年龄）比较三组基线特征，并用 Benjamini-Hochberg 法校正四项检验。年龄、既往用药史和初始抑郁程度未观察到显著组间差异。婚姻状况总体检验显著，但91.5%的卡方贡献来自仅见于B组的70例“未知”；剔除未知后，实质婚姻类别差异不显著（χ²=9.810, p=0.133, Cramér's V=0.040）。这表明婚姻信息存在明显的组间差异性缺失或记录流程异常，但现有数据不能识别为MNAR，也不能证明三组具有因果意义上的可交换性。医院×组别分配未见差异（χ²=0.001, p=0.9997），仅说明未观察到医院构成失衡，不能“排除”所有医院相关混杂。

## 七、后续可能要注意的点（给做问题2/3的同学）

1. 婚姻状况的"未知"分布不均，如果问题2要分析"婚姻状况对疗效的影响"，B组这70人怎么处理需要提前商量好（排除 / 单独归一类 / 做敏感性分析），别几个人各做各的口径不一样。
2. 附件1里"抑郁程度"在二院有部分缺失（约42人），代码里同样按"未知"处理，问题2用到这个变量时要沿用同一套缺失值处理逻辑，保持前后一致。
3. 二院组1只有524人（比其他组少1人），属于原始数据的情况，不是我们哪步筛漏了，写报告时如果有人问起可以直接解释。
