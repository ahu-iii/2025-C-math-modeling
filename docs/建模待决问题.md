## 1. 缓解的判据：D_12=0（单点）还是 D_6=D_12=0（连续两点）？——【已定案，见 ADR-0003】

- **结论**：**取单点 D_12 = 0**。这不是"选了个简便的默认"，而是被结构性理由锁定：**两点判据 D_6=D_12=0 恰是 ADR-0002 落地指标「豁免后持续不适」(D_6=1 或 D_12=1) 的逐个体补集**（互补性核验为 True，Pearson **r = −1.0000**；两点缓解率 + 豁免后持续不适率 = 100.00%，三组皆然）。ADR-0012 把 B_12 剪出指标集的理由正是"同轴指标会被重复计数、TOPSIS 假定各准则正交"——两点判据会以该缺陷的**极端形式**复活它：把 X 与 1−X 同时作为 4 个准则中的 2 个，等于把这根轴的权重偷偷翻倍到 50%。而两个指标都不能删（缓解率是题目点名的头号指标，豁免后持续不适率是 ADR-0002 这一**题目得分点**的唯一落地形式）。故单点是使 ADR-0012 指标集自洽的唯一选择。
- **本条不是关键路径**（原判断已被数据推翻）：12 组口径全跑，**B 排第一 12/12**，两点判据下缓解率 χ² p=0.0571 仍不显著、BH 后最小 p_BH=0.257 无指标转显著。换口径只改缓解率的绝对数值（C 94.66% → 87.51%），不改任何结论。
- **论文纪律**：不得写成"为简便起见取单点"——那把一个有结构性依据的决策讲成了偷懒。

## 2. 复发的"曾缓解"判据：D_3=0 或 D_6=0（宽松）还是仅 D_6=0（严格）？——【已定案，见 ADR-0004】

- **结论**：**取宽松版（D_3=0 或 D_6=0）**，与第 1 条联动定案。理由：ADR-0003 把缓解定为"**某单个时点无不适**"，"曾缓解"应是**同一把尺子量更早的窗口**（第 3、6 月）。严格版等于对"曾缓解"用比"缓解"本身更严的标准——第 12 月的单次无不适算缓解，第 3 月的单次无不适却不算数，这个不对称没有依据。
- **考虑过并否决的反向理由**：第 3 月落在 ADR-0002 的适应期窗口内，是否不该采信？不成立——ADR-0002 豁免的是**早期不适**（D=1 且后续消退者），从未主张第 3 月不是有效评估点；把"D_3=1 可豁免"读成"D_3=0 不算数"是对该规则的过度外推。
- **不改变结论**：宽松/严格/两点三种判据下 B 均排第一（全空间第一比例 99.9%/100.0%/100.0%）。

## 3. 复发率的分母：曾缓解人数 vs 全体受试者？——【已定案，见 ADR-0004】

- **结论**：**分母 = 曾缓解人数**（维持默认）。这是临床文献的标准定义（conditional on remission），建模方案.md 2.4 节已加粗强调"分母是缓解者，不是全体，务必注意"。**同时并列报告"全体口径"的数值**作为辅助描述指标，以防读者用另一口径复核时对不上数。
- **分母口径不是文字游戏**：曾缓解者占全体 92.9%（C）/93.0%（A）/95.7%（B），但在第 12 月仍不适的人里只有约 54% 曾缓解——持续不适者显著更少经历过缓解，故两种口径并非只差一个常数因子。
- **不改变结论**：两种分母下 B 均排第一（全空间第一比例 99.9% / 98.7%）。
- **论文纪律**：必须显式写出"分母为曾缓解者 n = 975/977/1005，非全体"。

## 4. "失访"是否要以某种方式计入 D_t 或单独建模？——【已定案，维持默认】

- **结论**：维持默认（失访不计入 D_t，按现有记录直接算，局限性中说明），**无需补充敏感性分析**。
- **依据**：本条要求的"失访按组分布"核查已跑：任何时点失访率 C 3.3%（35/1049）／A 4.2%（44/1050）／B 3.1%（33/1050），组间卡方 **p=0.3863，三组均衡**。故本条担心的"不良反应者提前失访导致某药看似疗效好"的选择偏倚没有证据支持，且失访率绝对水平低（3–4%）。加之题目问题3 明文"不需要考虑数据缺失和失访对评估的影响"，默认口径成立。
- **仍需在论文局限性中保留一句**：失访按 ADR-0001 不计入 D_t，三组均衡故未做插补；未观测到差异性失访的证据，但 3–4% 的失访仍是结论的一个不确定来源。

## 5. 初始抑郁程度在问题2 回归里怎么编码：有序变量（1/2/3）还是哑变量（dummy）？

- **当前默认建议**：有序变量，赋值 轻度=1 / 中度=2 / 重度=3，作为连续型协变量放入回归（或用有序Logistic专门处理）。
- **权衡/理由**：有序编码假设"程度每增加一级，效应大小近似线性/单调"，模型更简洁、系数更好解释（一个OR就能讲完"程度每加重一级，结局风险变化多少"），但如果实际关系不是单调的（比如中度反而比轻度和重度都特殊），有序编码会掩盖这种非线性；哑变量编码（以轻度为参照，中度、重度各一个哑变量）更灵活，能捕捉非单调关系，但需要额外确认参照组、系数数量翻倍、解释成本上升。建模方案.md 原始定义里已注明"有序，可赋 1/2/3"，倾向性较明确，但未排除哑变量作为稳健性检验的可能。
- **谁需拍板**：编程者可先用有序编码出主结果，若时间允许，用哑变量编码做一次稳健性对照，二者结论一致则更有说服力；是否值得花时间做这个对照，由全队根据时间紧张程度决定。

## 6. 症状负担 B_t 的数据来源：逐症状栏重新计数 vs 附件2"是否出现不适症状"栏？——【已定案】

- **结论（2026-07 问题2 实现中核实）**：两种来源在 **7 类实质症状**口径下**完全等价**（四时点零误差），任选其一皆可；且核实反证**失访本就不计入负担**（含失访重新计数与现成栏不符）。故 B_t = 7 类实质症状之和、失访排除。详见 ADR-0005"数据核实结论"。
- ~~当前默认建议：逐症状栏重新计数（8 类）~~ —— 已修正为 7 类（失访排除），并已确认与现成汇总栏等价。

## 7. TOPSIS 权重来源：纯熵权法（客观）还是熵权+专家权重结合？——【已定案，见 ADR-0012】

- **结论**：**两个分支都不选**。默认建议（纯熵权）已被数据否决：在真实数据上，纯熵权给题目点名的头号指标"缓解率"赋权 **0.0006**（剪枝后 0.0011），给每组仅 12–20 例的"自杀倾向率"赋权 **0.3465**（剪枝后 0.5864）——本条原先担心的"安全性指标被低估"没有发生，发生的是同一病根的反向爆掉：3 个备选估不出有意义的离散度，熵权度量的"能否分开三药"与"是否重要/可靠"无关。专家校正分支也不取：无题目提供的专家来源，且校正幅度足以反转结论时无法自证不是凑答案。
- **改定**：**主口径 = 等权**（外生、透明、无可调参数，是专家分支在"所有指标同等重要"下的极限情形）；**熵权保留为敏感性对照**并在论文中报告其失效原因；ADR-0007 要求的**权重敏感性分析**具体化为 **Dirichlet 单纯形全空间扫描**（20000 组权重：B 排第一占 99.9%）。指标集剪枝为 4 项（缓解率／复发率／豁免后持续不适率／自杀倾向率），ΔB 因基线依赖、B12 因与缓解率同轴重复计数而出局。
- **结论口径**：**B 优于 A 和 C（对权重稳健）；A 与 C 不可区分、并列**——A/C 次序随权重方案翻转，且所有组间检验均不显著。不得报告"B > A > C"全序。

## 8. 婚姻状况"未知"类别（集中在B组的70人）在问题2 回归里怎么处理？

- **当前默认建议**：保留为独立的"未知"类别参与回归（与问题1 一致的处理方式），不删除样本。
- **权衡/理由**：保留"未知"类别不损失样本量，且问题1 已证明该缺失非随机（集中在B组）——如果问题2 直接删除这些人，等于人为改变B组（药物B）的样本构成，可能引入选择偏倚，扭曲"药物B疗效"的估计。缺点是"未知"类别本身没有实际信息，其回归系数不好解释，且如果B组"未知"人数较多，可能拉低该类别系数的估计精度。可选的稳健性检验：单独排除"未知"样本重新跑一次婚姻状况这一项，对比结论是否一致。
- **谁需拍板**：编程者按默认口径先出结果；是否需要额外做"排除未知样本"的稳健性对照，由全队根据时间决定（问题1_实现说明.md 第七节已经提醒了这一点，说明队内已经意识到需要提前商量）。

## 9. 适应期豁免规则要不要影响症状负担 B_t 的数值，还是只影响"是否计入药物质量问题"的判定？

- **当前默认建议**：只影响不良反应发生率等"是否计入药物质量问题"类指标的判定，不改动 B_t 原始数值。
- **权衡/理由**：不改动 B_t 保持指标定义单纯、便于交叉核对（B_t 就是"当时观测到的症状数"，不掺入判断逻辑）；但这样"症状改善程度 ΔB=B_1-B_12"这个指标里，适应期个体前期的不适仍然会被计入 B_1，可能拉高 ΔB（看起来"改善"更多），这本身其实和适应期规则想表达的意思是一致的（适应期个体确实是"前期有事、后期没事"），所以两种指标（是否计入质量问题 vs B_t数值）目前看是不冲突的，但需要队内确认这个理解是否一致，避免论文里两处表述打架。
- **谁需拍板**：全队快速确认一下逻辑是否自洽即可，预计不需要长时间讨论。

## 10. 问题2 是否需要补充一次正式的"因素效应是否因药物而异"的检验？

- **当前默认建议**：不补充，仅做三组分层模型系数的描述性对比（见 ADR-0006）。
- **权衡/理由**：题目约束下无法在同一模型里放入"药物×因素"交互项（组别与药物完全共线），分层模型是唯一可行的主分析方式；但分层模型只能"描述性地"说"A组里这个因素的OR是2.1，B组是1.3，看起来不太一样"，无法给出一个正式的p值来说"这个不一样在统计上是否显著"。如果评委/论文需要更严谨的表述，可以考虑用其他方式间接检验（比如把三组数据汇总，去掉与组别共线的"药物主效应"，只放"因素×组别"交互项，专门检验交互项是否显著），但这会增加工作量。
- **谁需拍板**：全队评估这个"锦上添花"的检验值得不值得做，若时间紧张可以直接跳过、只做描述性对比，并在论文局限性里说明。
