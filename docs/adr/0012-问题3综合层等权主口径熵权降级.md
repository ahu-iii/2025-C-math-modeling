# 0012. 问题3综合评价：等权 TOPSIS 与两类不确定性

## 状态

已修订并接受（2026-07-17；取代 ADR-0007 的综合层）

## 背景

问题3需要把方向不同的多个主诉代理指标合成一个描述性排序。原方案用熵权法给三个药物赋权，但这里只有 3 个备选，熵权主要反映三行数据的离散程度，不能代表指标的重要性或估计可靠性。

两种常见预处理给出的缓解率权重分别约为 0.0011 和 0.4532，相差 423 倍；这说明熵权结果主要受预处理支配，不宜作为主口径。

## 决策

1. 主口径采用等权 TOPSIS。等权透明、可复现，不从 3×4 的小矩阵中估计“重要性”。
2. 熵权只作为方法敏感性对照，分别报告原始值和极差归一两种实现。
3. 综合评价保留 4 项：第12月无主诉率（效益型）、复发率、第6或12月仍有主诉率、自杀倾向率（后三项为成本型）。
4. 固定样本下用 Dirichlet 权重扫描评估**权重敏感性**。
5. 以受试者为抽样单位、在药物组内分层重抽样，评估**抽样不确定性**。两者必须分开解释。

`ΔB` 因依赖第1月水平而不进入综合层；第12月主诉负担与第12月无主诉率同轴，不重复计数；停药率事件较少且与持续主诉信息重叠，也不进入主矩阵。

## 固定样本结果

指标矩阵（%）：

| 药物 | 第12月无主诉率 | 复发率 | 第6或12月仍有主诉率 | 自杀倾向率 |
|---|---:|---:|---:|---:|
| C | 94.66 | 3.08 | 12.49 | 1.43 |
| A | 94.67 | 2.35 | 10.86 | 1.90 |
| B | 96.48 | 2.39 | 9.24 | 1.14 |

等权贴近度为 B 0.9768、A 0.4276、C 0.2601。Dirichlet 扫描 20,000 组权重时，B 在 99.9% 的权重组合中排第一。

该 99.9% 只回答：“在当前样本的指标值不变时，改变指标权重是否容易改变第一名？”它不包含受试者抽样误差，不能表述为 B 有 99.9% 的统计置信度。

当前样本中，B 在四项指标上都优于 C；因此在正权重下 B 的 TOPSIS 排名高于 C。这是**观测矩阵的确定性性质**，不是总体中的统计定理。

## 受试者重抽样结果

对每个药物组分别重抽受试者并重算四项指标与等权 TOPSIS，3,000 次结果为：

| 结果 | 估计 |
|---|---:|
| B 排名第一 | 92.5% |
| A 排名第一 | 5.7% |
| C 排名第一 | 1.8% |
| `P(score_B > score_A)` | 94.1% |
| `P(score_B > score_C)` | 98.0% |

B−A 的贴近度差 95% 重抽样区间为 `[-0.0924, 0.9790]`，包含 0；B−C 为 `[0.0162, 1.0000]`。因此可以说 B 呈现较一致的综合排名优势，尤其相对 C；不能说已证明 B 优于 A 和 C。

## 解释边界

- TOPSIS 是描述性决策汇总，不替代逐指标检验或纵向模型。
- 固定样本权重扫描与参与者重抽样回答不同问题，不得混称“稳健性”。
- A/C 次序会随权重和抽样变化，不报告完整全序。
- 操作性终点的定义先由数据含义决定，再检查综合指标是否重复；综合排名不能反向规定“缓解”的定义。
