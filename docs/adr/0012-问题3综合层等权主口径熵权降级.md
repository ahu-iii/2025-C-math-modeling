# 0012. 问题3 综合层：等权为主口径，熵权法降级为敏感性对照；指标集剪枝

## 状态
已接受（取代 ADR-0007 的"综合层"部分；ADR-0007 的"轨迹层"继续有效）

## 背景
ADR-0007 定的综合层是"熵权法客观赋权 → TOPSIS 排序 → 权重敏感性分析"，`docs/建模待决问题.md` open-Q7 把"纯熵权 vs 熵权+专家校正"列为待拍板项，默认建议纯熵权。

在真实数据上跑通后，纯熵权法失效，且不是程度问题而是方向性错误：

| 指标 | 熵权（7 指标） | 熵权（剪枝后 4 指标） |
|---|---|---|
| 缓解率 | **0.0006** | **0.0011** |
| 自杀倾向率 | **0.3465** | **0.5864** |
| 末次症状负担 B12 | 0.2468 | —（已剪枝） |
| 停药率 | 0.1430 | —（已剪枝） |
| 复发率 | 0.1256 | 0.2124 |
| 豁免后持续不适率 | 0.1183 | 0.2001 |
| ΔB 均值 | 0.0192 | —（已剪枝） |

题目正文点名的头号指标"缓解率"被赋权 0.0006（剪枝后 0.0011），数值上等于从模型中删除；而每组仅 12–20 例的"自杀倾向率"独占 35%（剪枝后 59%）的权重。剪枝并不能修复它——反而更极端。

根因是方法与样本量的错配：熵权法度量的是**各指标在备选之间的相对离散度**，本题只有 3 个备选（C/A/B），三行数据估不出有意义的离散度；且离散度回答的是"哪个指标能把三药分开"，而不是"哪个指标重要"或"哪个指标可靠"。缓解率 94.66/94.67/96.48 的确分不开三药（χ² p=0.0782），熵权法据此判定它无信息——这是方法按设计工作，但结论在临床与题意上都站不住。

open-Q7 担心的是"安全性指标被熵权低估"，实际发生的是同一病根的两个方向同时爆掉：罕见事件因估计噪声大而被超额赋权，题目主指标因真实差异小而被清零。

### 决定性证据（2026-07-16 实现后补强，源自对抗式验证）

上面"缓解率权重 0.0011"这一条证据**单独拿出来是可被攻击的**：熵权法在预处理上有两种通行写法，上表用的是"直接对原始值按列和归一"，而不少教科书写法是"先方向统一 + 极差归一到 [0,1] 再和归一"。审阅者可以质疑"你熵权法实现漏了归一化步骤，所以才得出 0.0011"。

补齐第二种口径后，结论**不是"修好了"，而是更糟**：

| 指标 | 熵权-原始值口径 | 熵权-极差归一口径 | 分歧倍数 |
|---|---|---|---|
| **缓解率** | **0.0011** | **0.4532** | **423×** |
| 复发率 | 0.2124 | 0.1708 | 1.2× |
| 豁免后持续不适率 | 0.2001 | 0.1940 | 1.0× |
| 自杀倾向率 | 0.5864 | 0.1819 | 3.2× |

**同一个指标，在熵权法的两种通行预处理下，权重相差 423 倍——从"毫无价值"（0.0011）跳到"占决策的 45%"（0.4532）**；且 A/C 次序在二者间翻转（原始值口径 B>C>A，极差归一口径 B>A>C）。

原因是 n=3 下两种预处理各自产生**方向相反的伪影**：极差归一必然把 [min, mid, max] 拉伸到 [0, x, 1] 撑满全幅，无论原始差异是 2 个百分点还是纯噪声，都被放大成"满幅信号"；原始值口径则让 94.66/94.67/96.48 的相对离散度趋近 0、权重塌成 0。二者是同一病根（**3 个备选估不出有意义的离散度**）的两种表现。

**这条比原来的证据强得多**：它不依赖"0.0011 这个权重不合理"这种需要临床判断的论证，而是一个纯形式的自相矛盾——**一个方法的输出因一个任意的预处理选择而在两个极端之间横跳，说明它在此样本量下没有信号可提取**。故代码两种口径都实现、都报告（`entropy_weights(M, preprocess="raw"|"minmax")`）。

补记的由来：这条是实现完成后跑对抗式验证时，由"统计正确性"维度的审查者提出"entropy_weights() 漏了正向化/极差归一"而牵出的。该条以"缺陷"论被证伪（它不改变任何最终结论，且补齐后的版本同样是伪影），但它精准命中了**论证的脆弱处**——按"会不会算错数"的标准它不算缺陷，按"论证能不能被攻破"的标准它是真问题。故不按"已证伪"归档，而是据此补强 ADR。

## 决策

**1. 主口径改为等权（equal weights）**，在剪枝后的指标集上做 TOPSIS。等权是外生于 3×k 矩阵的、透明且不可调参的——没有"谁定的这个权重"的口子，也不引入 open-Q7 需要的专家判断。

**2. 熵权法保留，降级为敏感性对照**，并在论文中显式报告其权重表与失效原因。展示"我们跑了熵权法、发现它在 3 备选下把题目主指标清零、故不作主口径"，比直接套用熵权法更有说服力。

**3. 权重敏感性分析（ADR-0007 已要求）的具体方法定为：Dirichlet 单纯形全空间扫描。** 在权重单纯形上均匀采样 20000 组权重，统计各药排名第一的比例，即"排名对权重是否敏感"的直接答案，不依赖任选的若干套方案。

**4. 指标集剪枝为 4 项**，每项对应一条独立的临床轴：

| 指标 | 方向 | 轴 | 依据 |
|---|---|---|---|
| 缓解率（D12=0） | 效益型 | 疗效 | 题目正文点名 |
| 复发率（/曾缓解） | 成本型 | 稳定性 | 题目正文点名；ADR-0004 |
| 豁免后持续不适率（D6=1 或 D12=1） | 成本型 | 药物质量 | ADR-0002 适应期豁免的落地 |
| 自杀倾向率 | 成本型 | 安全红线 | 安全性下限 |

**出局的指标及理由：**
- **ΔB 均值**：基线依赖。C 的 ΔB 最高（0.514）仅因 C 的 B_1 最高（0.617 vs A 0.556 / B 0.529）——变化量奖励基线差的组，方向与"疗效好"相反。ΔB 保留在 §5.1 描述性表格中（ADR-0005 定义不变、Kruskal-Wallis 照做，p=0.4880），但不进综合排名。
- **末次症状负担 B12**：与"缓解率 = 1−P(D12=1)"是同一个 D12 的两种写法，TOPSIS 假定各准则正交，同轴指标会被重复计数。
- **停药率**：低发且与"豁免后持续不适率"同向重叠。

（缓解率与豁免后持续不适率仍有部分重叠——前者是 D12=0，后者含 D12=1；二者非同一变量但相关。保留两者是因为一个是题目点名指标、一个是 ADR-0002 得分点，重叠已知并在论文中说明。）

## 结果（决策依据，非事后报告）

剪枝后指标矩阵（%）：

| 药 | 缓解率 | 复发率 | 豁免后持续不适率 | 自杀倾向率 |
|---|---|---|---|---|
| C | 94.66 | 3.08 | 12.49 | 1.43 |
| A | 94.67 | 2.35 | 10.86 | 1.90 |
| B | 96.48 | 2.39 | 9.24 | 1.14 |

TOPSIS 贴近度：

| 药 | 等权（主口径） | 熵权-原始值口径 | 熵权-极差归一口径 |
|---|---|---|---|
| B | **0.9768** | **0.9849** | **0.9856** |
| A | 0.4276 | 0.2832 | 0.2836 |
| C | 0.2601 | 0.4996 | 0.1773 |

**关键发现：B 稳居第一，A 与 C 的相对次序不稳健。**
- Dirichlet 全空间扫描（20000 组权重）：**B 排第一占 99.9%**，A 占 0.1%，C 占 0。
- 但 A/C 次序随方案翻转：等权 A > C；熵权-原始值 C > A；熵权-极差归一 A > C；留一检验中"删除复发率"亦翻为 C > A。
- B 在三套权重下贴近度均 > 0.97，与 A/C 之间始终隔着 0.5 以上的空档。

这与统计检验一致——所有组间检验均不显著（缓解率 p=0.0782、复发率 p=0.5260、任何主诉 p=0.2357、ΔB p=0.4880），A 与 C 本就分不开，TOPSIS 的次序差异是噪声。**故结论定为：B 优于 A 和 C（对权重稳健）；A 与 C 不可区分，并列。** 不得报告"B > A > C"这样的全序。

### 结构性解释（2026-07-16 补强）：扫描结果不是模拟出来的巧合

上面的 99.9% / 0.1% / 0.0% 是扫描**测**出来的，但它们其实可以从指标矩阵直接**读**出来。补记于此，因为"能解释"比"跑出来"更经得起追问：

| | 缓解率 | 复发率 | 豁免后持续不适率 | 自杀倾向率 |
|---|---|---|---|---|
| **B vs C** | B 优 | B 优 | B 优 | B 优 |
| **B vs A** | B 优 | **A 优 0.04pp** | B 优 | B 优 |
| **A vs C** | A 优 0.01pp | A 优 | A 优 | **C 优** |

- **B 在全部 4 个指标上优于 C（Pareto 支配）**。方向统一后 B 的归一值逐列 ≥ C，故 D⁺ 更小、D⁻ 更大、贴近度必然更高——**B > C 对任意正权重成立，是定理而非统计结果**。扫描里 C 排第一占 `0.0%` 因此不是"四舍五入到 0"，而是恰好为 0。模拟与定理对上了，这本身是对实现的一次交叉验证。
- **B 对 A 只差一项**：复发率 2.39% vs 2.35%，差 0.04pp（p = 1.0000）。那 0.1% 的反超正是把权重几乎全压在复发率上的组合——A 的全部本钱是 0.04pp 的噪声。
- **A 与 C 的优劣方向相反**：A 的优势在复发率与豁免后持续不适率，C 的优势在自杀倾向率；缓解率上 A 只领先 0.01pp（1 个人），极差归一后仅值 0.0055，对 A/C 之争几乎不贡献。故次序取决于"自杀倾向率的权重能否压过另两项之和"：等权下 A 的复发率(归一=1.00)+豁免(0.50) 压过 C 的自杀(0.62) → A > C；熵权-原始值给自杀倾向率 58.6% 权重 → C 反超；留一检验删掉复发率后同样翻成 C > A。**三处翻转是同一个机制，不是三条独立证据。**

### 论证纪律：翻转是症状，不是论据

原文把"A/C 次序随方案翻转"当作"二者不可区分"的**直接证据**。这个说法有漏洞，且已被验证会出事：

- 在**两点缓解判据**（D₆=D₁₂=0，ADR-0003 已排除的备选）下重跑，**翻转消失**——等权、熵权-原始值、熵权-极差归一三套方案全部给出 B > A > C。
- 但此时 A 与 C 依然分不开：A vs C 的缓解率 p = 0.2733，四项指标两两检验全不显著，BH 校正后指标层最小 p_BH = 0.257。

**即：翻转是"不可区分"的充分条件，不是必要条件。** 若把论据挂在翻转上，换一套同样可辩护的口径就会凭空"升格"出一个 A > C 的全序——而它建立在纯噪声上。

故论证次序更正为：**A/C 不可区分的论据是"四项指标的 A vs C 检验全不显著（p = 1.0000 / 0.3991 / 0.2733 / 0.4971）且优劣方向相反"**；翻转是这一事实在 TOPSIS 上的表现，是佐证，不是地基。论文须按此次序写。

（本条与上面「决定性证据」同源：对抗式验证按"会不会算错数"的标准找不出缺陷，但按"论证能不能被攻破"的标准，这里确实有一处松动。）

## 备选方案
- **纯熵权法（open-Q7 默认建议）**：已被数据否决，理由见背景。
- **熵权 + 专家校正 / 权重下限（open-Q7 另一分支）**：需要有人定"自杀倾向率权重下限该是 0.15 还是 0.20"，无题目提供的专家来源可依，且校正幅度足以反转结论时无法自证不是凑答案。等权是该分支的极限情形（所有指标同等重要），但不需要任何主观参数，故取等权。
- **CRITIC 等其他客观赋权法**：同样从 3×k 矩阵取数，样本量病根不变，换汤不换药。
- **保留 7 指标不剪枝**：ΔB 的基线依赖会把"基线最差"错读为"改善最多"，B12 与缓解率重复计数，均属已知错误，不予保留。

## 影响/后果
- 论文综合层必须同时给出等权与熵权两套结果 + Dirichlet 扫描图，并写明熵权失效的原因；这一段从"套用方法"变成"方法选择的论证"，是加分点而非减分点。
- **结论口径收紧为"B 第一 + A/C 并列"**，问题4（ADR-0008）的选药建议不能再依赖 A 与 C 的先后，需相应调整。
- open-Q7 就此定案（不是"确认默认"，而是默认被否决、改选等权），可从 `docs/建模待决问题.md` 移除。
