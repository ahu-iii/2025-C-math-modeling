# 0004. 复发 / 复发率的定义（含分母口径）

## 状态
已接受（分母的具体口径为讨论重点，需团队最终确认，详见 `docs/建模待决问题.md`）

## 背景
复发是"先缓解后又出现症状"的动态过程，无法用单一时点判断，必须结合多个时点；且"复发率"的分母选取（占谁的比例）容易产生歧义——建模方案.md 中特别用加粗强调"务必注意"，说明这是一个容易被不同人做出不同口径的高风险点。

## 决策
个体层面：D_3 = 0 或 D_6 = 0（曾经缓解过）**且** D_12 = 1（末次又出现不适）即判定为复发；复发率 = 复发人数 / **曾缓解人数**（分母是"曾缓解者"，不是全体受试者）。

## 备选方案
- **分母用全体受试者**（复发率 = 复发人数 / 总人数）：概念上更接近"总体不良结局率"而非临床意义上的"复发率"，与文献中"复发率是以缓解为条件"的标准定义不一致，不采纳为主口径，但可作为辅助描述性指标保留。
- **分母严格限定为"第 6 月缓解者"**（而非"第 3 或第 6 月曾缓解"）：更严格、更贴近"从缓解状态复发"的单点定义，但会缩小分母人群、丢弃"第 3 月缓解、第 6 月已复发"的个体。宽松版（3 月或 6 月满足其一）与严格版（仅 6 月）尚未最终二选一，已列入待决问题。

## 影响/后果
分母选取直接决定复发率数值大小及三组间比较结果；论文和口头汇报必须明确写出分母定义，避免读者/评委误解为"占总受试者比例"。
