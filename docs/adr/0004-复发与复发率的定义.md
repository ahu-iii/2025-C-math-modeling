# 0004. 复发 / 复发率的定义（含分母口径）

## 状态
已接受（2026-07-16 定案，open-Q2 与 open-Q3 一并关闭；与 ADR-0003 联动决定）

## 背景
复发是"先缓解后又出现症状"的动态过程，无法用单一时点判断，必须结合多个时点；且"复发率"的分母选取（占谁的比例）容易产生歧义——建模方案.md 中特别用加粗强调"务必注意"，说明这是一个容易被不同人做出不同口径的高风险点。

原先留了两个未拍板的分支：
- **open-Q2**：曾缓解 = D_3=0 或 D_6=0（宽松） vs 仅 D_6=0（严格）。
- **open-Q3**：复发率分母 = 曾缓解人数 vs 全体受试者。

`docs/建模待决问题.md` 强调本条与 open-Q1（缓解判据）"必须联动决定，不能分开定"。ADR-0003 已把缓解判据锁定为**单点 D_12 = 0**，本 ADR 据此定案。

## 决策
个体层面：**D_3 = 0 或 D_6 = 0（曾经缓解过）且 D_12 = 1（末次又出现不适）** 即判定为复发；**复发率 = 复发人数 / 曾缓解人数**（分母是"曾缓解者"，不是全体受试者）。

即：open-Q2 取**宽松**、open-Q3 取**曾缓解人数**——两条均维持原默认，但依据已从"建模方案的倾向性表述"升级为下述理由。

## 决策依据

### open-Q2：宽松版与 ADR-0003 的单点判据同构

ADR-0003 定的缓解判据是"**某单个时点无不适**"（D_12 = 0）。"曾缓解"应当是**同一把尺子量更早的窗口**，而非另一把尺子：可用的更早时点是第 3、6 月，故"曾缓解 = D_3 = 0 或 D_6 = 0"。

严格版（仅 D_6 = 0）等于对"曾缓解"用**比"缓解"本身更严**的标准——第 12 月的单次无不适足以判为缓解，第 3 月的单次无不适却不算数。这个不对称没有依据：两者都是单次随访评估，凭什么第 3 月的观测不作数。

（考虑过一条反向理由：第 3 月落在 ADR-0002 的适应期窗口内，是否不该采信？不成立——ADR-0002 豁免的是**早期不适**（D=1 且后续消退者），它从未主张第 3 月不是有效评估点。把"D_3=1 可豁免"读成"D_3=0 不算数"是对该规则的过度外推。）

### open-Q3：分母 = 曾缓解人数

`docs/建模方案.md` 2.4 节已加粗强调"分母是缓解者，不是全体"，这是临床文献中"复发率"的标准定义（conditional on remission）。本条实质上早已定案，此处正式确认。

值得记一笔的是分母**确实**改变数值、且改变方向合乎预期：全队列中曾缓解者占 92.9%（C）/ 93.0%（A）/ 95.7%（B），但在第 12 月仍不适的人里只有约 54% 曾缓解——即持续不适者显著更少经历过缓解。故"复发率"（条件概率）与"全体中复发的比例"并非只差一个常数因子，分母口径不是文字游戏。

### 结论对两条口径均不敏感

`definition_robustness()` 把 缓解判据(2) × 曾缓解判据(3) × 复发率分母(2) = **12 组口径全跑**（输出见 `output/problem3_definition_robustness.csv`）：

| 曾缓解判据 | 分母 | 复发率 C/A/B % | 等权排序 | B 全空间第一 |
|---|---|---|---|---|
| 宽松 D_3=0或D_6=0 | 曾缓解（**主口径**） | 3.08 / 2.35 / 2.39 | B > A > C | 99.9% |
| 宽松 | 全体 | 2.86 / 2.19 / 2.29 | B > A > C | 98.7% |
| 严格 仅 D_6=0 | 曾缓解 | 2.65 / 1.89 / 1.65 | B > A > C | 100.0% |
| 严格 | 全体 | 2.38 / 1.71 / 1.52 | B > A > C | 100.0% |
| 两点 D_3=D_6=0 | 曾缓解 | 2.49 / 1.73 / 1.50 | B > A > C | 100.0% |
| 两点 | 全体 | 2.00 / 1.43 / 1.24 | B > A > C | 100.0% |

**B 排第一在 12/12 组口径下成立**，全空间第一比例 97.9%–100.0%。复发率的绝对数值随口径在 1.24%–3.08% 间变化，三药次序与最终结论均不变。

## 备选方案
- **曾缓解取严格版（仅 D_6=0）**：对"曾缓解"用比"缓解"更严的尺子，不对称且无依据（见上）。数值上不改变结论，故非错误、仅非首选。
- **曾缓解取两点版（D_3=D_6=0）**：是 ADR-0003 两点缓解判据的同构版本；既然缓解判据已锁定为单点，本条采用两点即破坏同构，排除。
- **分母用全体受试者**：与文献标准定义不一致，不作主口径；**保留为辅助描述性指标**在论文中并列报告（数值见上表"全体"行），以防读者用另一口径复核时对不上数。

## 影响/后果
- 复发率口径就此固定，ADR-0012 综合层的第二列输入不再是"暂用默认"。
- 论文与口头汇报必须显式写出分母定义（"分母为曾缓解者 n = 975/977/1005，非全体"），避免评委误读为占总受试者比例；同时并列给出全体口径的数值。
- 复发率的三组差异不显著（p = 0.5260），且在综合层中是 A 相对 B 的唯一优势项（A 2.35% vs B 2.39%，差 0.04pp，p = 1.0000）——这 0.04pp 正是权重全空间扫描中 A 仅有的 0.1% 反超所依赖的全部本钱。论文不得据此称"A 在复发率上优于 B"。
