# 0009. 问题1：大样本 Kruskal-Wallis 替代 ANOVA、卡方+蒙特卡洛兜底、BH 校正、缺失记为"未知"

## 状态
已接受（已实现，见 `docs/问题1_实现说明.md` 及 `code/problem1_baseline.py`）

## 背景
问题 1 需检验三组（药物 C/A/B）在年龄、婚姻状况、既往用药史、初始抑郁程度四个基线变量上是否均衡，作为后续三问"疗效差异可归因于药物而非人群底子不同"这一论文主线的前提。数据量大（每组 n>1000）且存在缺失，常规检验方法在此规模和缺失模式下需要额外处理。

## 决策
- **年龄**：先做 Shapiro-Wilk 正态性 + Levene 方差齐性诊断，大样本下自动切换为 Kruskal-Wallis（而非 ANOVA），因为大样本下 Shapiro-Wilk 几乎总会判定"不服从正态"（该检验对样本量敏感是已知特性），直接用 ANOVA 会违反其前提假设。
- **分类变量**（婚姻状况/既往用药史/初始抑郁程度）：卡方独立性检验；期望频数 < 5 的格子存在时，额外跑蒙特卡洛置换检验（随机打乱组别标签 2000 次）交叉验证兜底。
- **多重比较**：四个基线变量的检验结果一起做 Benjamini-Hochberg（FDR）校正。
- **缺失处理**：分类变量缺失不删除样本，记为"未知"类别参与检验。

## 备选方案
- **年龄坚持用 ANOVA 而不切换**：违反正态性前提，结论不可靠，未采纳。
- **分类变量缺失直接删除（完整病例分析）**：会丢样本量，且婚姻状况缺失集中在 B 组；删除会改变 B 组样本构成并可能引入选择偏倚，未采纳。现有数据只能识别组间差异性缺失，不能判定缺失概率是否依赖未观测的真实婚姻状态（MNAR）。
- **不做多重比较校正**：四次检验同时用 α=0.05 会累积假阳性风险，未采纳。

## 影响/后果
结论为年龄/既往用药史/初始抑郁程度三组无显著差异（FDR 校正后 p 分别为 0.849、0.928、0.849）；婚姻状况表面显著（χ²=152.871, p<0.001），经三步定量排查证实系 B 组 70 人婚姻状况缺失集中导致，并非真实人群差异：

1. **效应量诊断**：Cramér's V=0.156，仅"小到中等"，与 p≈5×10⁻²⁹ 的极度显著不匹配 → 大样本敏感性信号。
2. **卡方贡献度分解**：总卡方的 **91.5%**（139.93/152.87）由"未知"一列单独贡献，四个实质类别合计仅 8.5%。
3. **剔除敏感性分析**：移除"未知"后 χ²=9.810, df=6, **p=0.133（不显著）**，Cramér's V 降至 0.040。
4. **缺失模式检验**：缺失×组别 χ²=143.115, df=2, p=8.4×10⁻³² → 缺失率与组别强相关，不符合简单 MCAR 图景；但不能据此识别为 MNAR。

这些结果支持“除记录缺失异常外，未观察到明显基线失衡”的描述，但基线检验不显著不能证明随机化、可交换性或因果归因成立。
