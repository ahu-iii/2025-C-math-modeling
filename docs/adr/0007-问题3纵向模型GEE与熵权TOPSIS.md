# 0007. 问题3：GEE/混合效应模型处理 4 时点重复测量 + 熵权法-TOPSIS 综合排名

## 状态
已接受

## 背景
问题 3 需利用"每人 4 个时点（1/3/6/12 月）"的纵向结构回答"用药前后变化"及"哪种药改善更快"；单纯逐时点独立比较（如每个时点单独跑卡方）会忽略同一受试者多次观测之间的相关性，低估标准误、可能夸大显著性。同时问题 3 需要把缓解率、复发率、症状改善、不良反应率等方向不一致（有的越高越好、有的越低越好）的多个指标合成一个总排名供问题 4 使用，需要一个客观、可复现的赋权与排序方法。

## 决策
- **轨迹层**：用广义估计方程（GEE）或混合效应 Logistic 模型，设定 P(D_t=1) ~ 药物 + 时间 + 药物×时间，用交互项刻画"不同药物随时间下降速度是否不同"，直接回答"哪种药改善更快"。
- **综合层**：用熵权法（客观赋权，权重由各指标本身的区分度决定，不依赖主观打分）确定各指标权重，再用 TOPSIS 计算三药与理想解的贴近度并排序，并做权重敏感性分析验证排名稳健性。

## 备选方案
- **轨迹层用重复测量 ANOVA 替代 GEE/混合模型**：要求结局连续且满足球形假设，D_t 是二元变量不适用，未采纳；可选补充 Kaplan-Meier + Log-rank 做"达到缓解速度"的生存分析，作为加分项而非必选项。
- **综合层用专家打分/层次分析法（AHP）主观赋权替代熵权法**：更贴近临床经验，但依赖专家主观判断、复现性差，且题目未提供专家意见来源，故选熵权法作为客观主口径；权重具体来源（纯熵权 vs 熵权+专家校正）仍是一个待队内确认的点，已列入待决问题。

## 影响/后果
GEE 需要选择合适的相关结构（如可交换相关），若不同药物间的时间趋势差异未达显著，"哪种药改善更快"这一结论需谨慎表述为"未观测到显著差异"而非"没有差异"；熵权法权重完全由数据分布决定，若某指标区分度很小可能被赋予很低权重，需在论文中列出具体权重值供读者判断合理性。
