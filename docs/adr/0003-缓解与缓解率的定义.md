# 0003. 缓解 / 缓解率的定义

## 状态
已接受（个体判据的严格程度尚有一个分支未最终拍板，详见 `docs/建模待决问题.md`）

## 背景
问题 3 要求"综合评估三种药疗效，指标包括缓解率"，但附件2 没有现成的"缓解"标记，需要队内从 D_t 统一构造，避免各人代码口径不一致导致结果对不上。

## 决策
个体层面：D_12 = 0（第 12 月无不适）即判定为缓解；缓解率 = 某组缓解人数 / 该组总人数。

## 备选方案
更严格的版本要求 **D_6 = D_12 = 0**（连续两个时点都无不适才算缓解），排除"第 6 月还有症状、第 12 月恰好不适消失"的偶然波动，更贴近临床"持续缓解"的含义，但会降低缓解率数值、且对短期波动更敏感。建模方案.md 中两种定义并列给出，未最终二选一，已放入待决问题清单，留待团队拍板。

## 影响/后果
缓解定义直接影响缓解率、复发率（缓解率是复发率分母的基础）、以及问题 3 熵权-TOPSIS 综合排名的输入指标；一旦切换定义，需要重新计算全部下游指标并复核结论是否稳健。
