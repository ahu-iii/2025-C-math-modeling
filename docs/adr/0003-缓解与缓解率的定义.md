# 0003. 缓解 / 缓解率的定义

## 状态
已接受（2026-07-16 定案，open-Q1 关闭；单点判据不再是"默认"，而是被结构性理由锁定）

## 背景
问题 3 要求"综合评估三种药疗效，指标包括缓解率"，但附件2 没有现成的"缓解"标记，需要队内从 D_t 统一构造，避免各人代码口径不一致导致结果对不上。

原先本 ADR 留了一个未拍板的分支（open-Q1）：单点 **D_12 = 0** vs 连续两点 **D_6 = D_12 = 0**。`docs/建模方案.md` 2.3 节两种并列给出、未二选一，理由是"两点更贴近临床'持续缓解'的严谨含义"。

## 决策
个体层面：**D_12 = 0（第 12 月无不适）即判定为缓解**；缓解率 = 某组缓解人数 / 该组总人数。

**两点判据 D_6 = D_12 = 0 不是"更严格的可选项"，而是与 ADR-0012 的指标集不相容，必须排除。**

## 决策依据

### 1. 两点判据与「豁免后持续不适率」是同一个变量的正反两面（决定性）

ADR-0002 落地出的指标「豁免后持续不适」定义为 **D_6 = 1 或 D_12 = 1**。其逻辑否定恰好是 **D_6 = 0 且 D_12 = 0** —— 即两点缓解判据本身。二者互为补集，不是"相关"而是**恒等**：

| 药 | 两点缓解率 | 豁免后持续不适率 | 合计 |
|---|---|---|---|
| C | 87.51% | 12.49% | 100.00% |
| A | 89.14% | 10.86% | 100.00% |
| B | 90.76% | 9.24% | 100.00% |

逐个体核验互补性为 `True`，Pearson **r = −1.0000**（单点判据下 r = −0.6385，两者仍是两条不同的轴）。

ADR-0012 把「末次症状负担 B_12」剪出指标集，理由是"与缓解率是同一个 D_12 的两种写法，TOPSIS 假定各准则正交，同轴指标会被重复计数"。**两点判据会以该缺陷的极端形式复活它**：把 X 与 1−X 同时作为 4 个准则中的 2 个喂给 TOPSIS，等于把这一根轴的权重悄悄翻倍到 50%。两个指标又都不能删——「缓解率」是题目正文点名的头号指标，「豁免后持续不适率」是 ADR-0002 适应期豁免这一**题目得分点**的唯一落地形式。故单点判据是使 ADR-0012 指标集自洽的唯一选择。

（旁证：两点判据下，指标层里「缓解率」与「豁免后持续不适率」的 χ² p 值同为 0.0571，一字不差——因为它们本就是同一个检验。）

### 2. 结论对本条口径不敏感（故本条不是结论的"关键路径"）

先证明换口径不改变结论，再按上述结构性理由在其中挑一套。`definition_robustness()` 把 缓解判据(2) × 曾缓解判据(3) × 复发率分母(2) = **12 组口径全跑**：

- **B 排第一：12/12**；B 在权重全空间中排第一的比例 **97.9%–100.0%**。
- 缓解率数值随口径变化不小（C 94.66% → 87.51%），但三药的**次序**（B 最高）在两种判据下都不变。
- 两点判据下缓解率三组 χ² p = 0.0571，仍不显著；BH 校正后指标层最小 p_BH = 0.257，无指标转显著。结论"药物间未观测到显著差异"不变。

原先 `docs/建模待决问题.md` 把本条列为"问题3 的关键路径、建议优先拍板"——**该判断已被数据推翻**：它不改变任何结论，只改变缓解率的绝对数值。

## 备选方案
- **两点 D_6 = D_12 = 0**：见上，与「豁免后持续不适率」恒等互补，破坏 ADR-0012 指标集的正交性假定，排除。若将来有人坚持采用，必须同时把「豁免后持续不适率」剪出综合层——那会丢掉 ADR-0002 这个题目得分点在综合层的唯一体现，代价更大。
- **三点/全程无不适**：数据仅 4 个时点，且第 1、3 月受适应期反应支配（ADR-0002），要求全程无不适等于把适应期个体全判为未缓解，与 ADR-0002 直接冲突。

## 影响/后果
- 缓解率、复发率（缓解是其分母的基础）、ADR-0012 综合排名的输入就此固定，下游可放心引用，不必再标注"暂用默认口径"。
- 论文写作**不应**把单点判据说成"为简便起见选择单点"——那会把一个有结构性依据的决策讲成偷懒。正确表述：两点判据与适应期豁免指标恒等互补，会在综合层重复计数，故取单点。
- 缓解率仍与「豁免后持续不适率」部分相关（r = −0.6385，两者都含 D_12），此重叠 ADR-0012 已知悉并在论文中说明；单点判据把它控制在"相关"而非"恒等"。
