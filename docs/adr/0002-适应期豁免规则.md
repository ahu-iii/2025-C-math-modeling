# 0002. 适应期豁免规则

## 状态
已接受

## 背景
题目原文明确规定："受试者前期（第 1、3 月）出现不适、但后期（第 6、12 月）不适消失，视为已适应该药物，此时前三个月的不适不计入药物质量问题。" 这是**题目强制规定的评分点**，不落实会直接影响得分，也是本题区别于普通临床数据分析的关键规则。

## 决策
形式化为：若 D_1 = 1 或 D_3 = 1，且 D_6 = 0 且 D_12 = 0，则判定该受试者为"适应期反应"；在评价药物质量/疗效（尤其是不良反应发生率等指标）时，对其前三个月的不适做豁免处理，不计入"药物质量问题"的统计。

## 备选方案
未提出实质性备选方案——这是题目的硬性定义，唯一的自由度在于"豁免范围要不要延伸到症状负担 B_t 的数值本身"（即豁免后 B_1、B_3 是否也要相应调低）。当前默认只在"是否计入药物质量问题"的判定（如不良反应发生率）中豁免，不改动 B_t 本身的原始数值，避免定义之间过度耦合、增加理解成本。

## 影响/后果
直接影响问题 3 的不良反应发生率、缓解率等指标的具体计算口径；评委会重点核查是否落实该规则，论文方法部分必须显式写出上述公式化定义，不能只用文字描述。
