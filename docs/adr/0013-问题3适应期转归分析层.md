# 0013. 问题3 新增"适应期转归分析"层（限定 D_1=1 子群）

## 状态
已接受（在 ADR-0007 的"指标层/轨迹层/综合层"之外新增一层）

## 背景
ADR-0007 为问题3 规划了三层：指标层（逐指标组间检验）、轨迹层（GEE）、综合层（TOPSIS）。三层在真实数据上跑通后，**全部组间比较均不显著**：

| 检验 | 结果 |
|---|---|
| 缓解率 χ²（94.66/94.67/96.48） | p = 0.0782 |
| 复发率 χ²（3.08/2.35/2.39） | p = 0.5260 |
| 任何主诉 χ² | p = 0.2357 |
| ΔB Kruskal-Wallis | p = 0.4880 |
| GEE 药物主效应（A: OR 0.907 / B: OR 0.871） | p = 0.283 / 0.124 |
| GEE 药物×时间 交互（连续 log 月，2 df 联合 Wald） | p = 0.1008 |
| GEE 药物×时间 交互（时间为分类，6 df 联合 Wald） | p = 0.5330 |

即 ADR-0007 规划的全部分析中，没有任何一项能把三药分开。轨迹层的强结论是**时间**而非**药物**：时间主效应 OR 0.47 / log 月（ρ=0.35，可交换相关），第 12 月不适几率约为第 1 月的 15%。

但在"限定第 1 月已出现不适者（D_1=1）"的子群里存在唯一显著的组间差异，且这个子群恰好就是 ADR-0002 适应期规则所描述的人群——"前期出现不适"者。ADR-0007 没有规划这一层，纯粹是因为规划在跑数之前。

## 决策
新增**适应期转归分析层**：限定 D_1=1 的受试者，分析其在第 3/6/12 月的不适转归，回答"早期出现不适者中，谁能适应过来"。

- **主分析**：D_1=1 子群中各时点仍不适率的组间 χ² 检验。
- **补充**：以"首次不适消退"（首个 D_t=0 的月份，末次仍不适者按第 12 月删失）为事件的 Kaplan-Meier 曲线 + Log-rank 检验，即 ADR-0007"备选方案"中列为可选加分项的生存分析——现在它有了明确的分析对象，从可选升为该层的正式补充。需引入 `lifelines` 依赖（当前未安装）。

**混杂控制说明**：D_1 是治疗后变量，按后随机化变量分层原则上有对撞偏倚风险。此处可接受，因为三组 D_1 发生率均衡（C 27.7% / A 25.8% / B 26.0%，n = 291/271/273），子群在三臂间是可比的；该限制在论文局限性中写明。

## 结果（决策依据）

D_1=1 子群中仍不适率（%）：

| 药 | n | 第3月 | 第6月 | 第12月 |
|---|---|---|---|---|
| C | 291 | 43.6 | 27.8 | **11.7** |
| A | 271 | 42.4 | 26.9 | **17.3** |
| B | 273 | 39.2 | 21.2 | **8.1** |
| χ² p | | 0.5452 | 0.1544 | **0.0041** |

始终未消退者（D_3=D_6=D_12=1）：C 24/291 = 8.2%、A 32/271 = 11.8%、B 12/273 = 4.4%。

这是全数据集中唯一显著的组间结果（p=0.0041），并揭示一个全队列指标看不到的机制差异：**A 引发早期不适的人数略少于 C（25.8% vs 27.7%），但 A 的不适更"黏"（第 12 月仍不适 17.3% vs C 11.7%）**，两者在第 12 月净持平（均 5.3%），是经由相反的机制到达同一个终点；B 在两条路径上都占优。

## 备选方案
- **不新增此层，仅作稳健性注脚**：会丢掉数据中唯一显著的组间结果，且丢掉问题4 唯一可用的机制性依据（见影响）。不采纳。
- **改以全队列做交互检验来捕捉同一现象**：已做，即 GEE 药物×时间 交互，p=0.1008 / 0.5330，检不出——因为 68% 的受试者全程无不适，在全队列中稀释掉了信号。这正是需要限定子群的理由。

## 影响/后果
- 问题3 的结论结构变为：**（1）三药均显著有效，改善由时间驱动；（2）综合排名 B 第一且对权重稳健，A 与 C 不可区分（ADR-0012）；（3）在早期出现不适者中，A 的转归显著最差、B 最好。** 第（3）条是唯一有 p<0.05 支撑的组间结论。
- 给问题4（ADR-0008）提供了可操作的分层建议依据：对"用药早期即出现不适"的患者，A 与 B/C 的差别是实质性的——这比 ADR-0012 那条"A/C 并列"的钝结论更有临床可用性。
- 需新增 `lifelines` 依赖（`pyproject.toml`）。
- 论文须写明按后随机化变量（D_1）分层的局限性，并附三组 D_1 发生率均衡的证据。

## 关联
ADR-0002（适应期规则，本层是其正面操作化）、ADR-0007（新增其未规划的一层；生存分析由"可选加分项"升为正式补充）、ADR-0012（综合层结论）、ADR-0008（问题4 分层建议依据）。
