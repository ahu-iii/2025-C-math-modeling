# 0005. 症状负担 / 改善程度的定义

## 状态
已接受（数据来源二选一尚待核实，详见 `docs/建模待决问题.md`）

## 背景
问题 3 要求"症状改善程度"作为连续型疗效指标。D_t 是二元变量（有/无不适），不足以体现"症状越来越轻"这种程度上的变化趋势，需要一个能刻画"不适严重程度"的量化变量。

## 决策
症状负担 B_t = 该时点 8 类症状栏中标记为 1 的种类数之和（每个时点算一个计数）；改善程度 ΔB = B_1 − B_12（正值代表改善），可选配合相对改善率 (B_1 − B_12) / B_1。

## 备选方案
直接采用附件2"是否出现不适症状"栏的数值作为 B_t：若该栏本身就是症状种类数的汇总，则与本定义等价，可省去重新计数的步骤；若该栏只是二元标记，则会退化为与 D_t 重复，不再具备"负担程度"的信息量。两种数据来源是否等价**尚未核实该栏的实际编码含义**，核实前暂用"逐症状栏重新计数"作为主口径，以确保至少有一个可用的连续型指标。

## 影响/后果
B_t 是问题 3 中唯一的连续/计数型疗效指标，会进入 Kruskal-Wallis/ANOVA 组间比较及熵权-TOPSIS 综合排名的其中一项输入；若后续核实两种数据来源不一致，需要重新计算并在论文中说明差异原因。
