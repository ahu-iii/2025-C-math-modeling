# 0001. "不适/主诉"单时点定义（D_t）

## 状态
已接受

## 背景
附件2 原始数据是"是否出现某类不适症状"的多列标记（8 类症状 × 4 个随访时点：第 1、3、6、12 月），没有现成的"某人在某时点算不算不适"的单一变量。但问题 2、3、4 的所有结局变量（缓解、复发、症状负担）都要先知道每个时点的不适状态，需要队内统一口径，否则各自代码算出的缓解率、复发率会对不上。

## 决策
定义 D_t（t ∈ {1,3,6,12} 月）：该时点"失眠、脱发、激素水平异常、嗜睡、便秘、有自杀倾向、副作用导致停药"7 项实质性不适症状中至少一项标记为 1，则 D_t = 1（该时点有不适），否则 D_t = 0。"失访"单独处理，**不计入** D_t（失访是缺失，不是症状）。

## 备选方案
- **直接采用附件2 现成的"是否出现不适症状"栏作为 D_t 来源**：省事，但该栏的具体统计口径不透明（是否包含失访、是否与逐症状栏完全等价未核实），存在风险，暂不作为主口径，仅作交叉核对用。
- **把失访也计为 D_t = 1**（视为"无法评估 = 最差结局"的保守处理）：题目未要求，且会人为拉高不适率、扭曲组间比较，未采纳。

## 影响/后果
所有下游指标（缓解率、复发率、症状负担）均建立在此定义之上；若后续改用附件2 现成栏位作为主口径，需要重新跑一遍问题 2/3 的全部结果并核对两种口径是否一致。
