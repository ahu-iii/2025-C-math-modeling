## 范文里我们照搬了什么

`docs/论文初稿.md` 开头的"插图排版"与 `docs/adr/0011` 均以此范文为依据，具体是：

- **图题在图下方**、加粗、格式为 `图N␣␣图题`（图号与图题间空两格，题末不加句号）——见范文 p.8 图1、p.9 图2；
- **图号全文连续**，且按正文首次引用的先后排列，不按章节分组——范文 图1–图7 严格如此；
- **正文以"图N ……"引用**，不写文件名——如范文 p.6"图 1 给出清洗后发电量的数据分布图"；
- **每张图后必有解读**——范文 p.6、p.16 专设"结果分析"逐图讲"看到了什么、说明什么"。

需要核对体例时看范文 pp.6–16，那几页图最密集。
